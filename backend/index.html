Api search image

Hello, this app will help you to find incredibly beautiful images

To find images you need to specify the following addresshttps://api-search.glitch.me/images/[YOUR SEARCH DATA]?offset=1

After searching you'll take the images address, page url, active search and previous search in the end of the list.

On the page you'll find average 20 images.
If you want to see more images, just specify the following page like that https://api-search.glitch.me/images/[YOUR SEARCH DATA]?offset=[INTEGER NUMBER PAGE]
